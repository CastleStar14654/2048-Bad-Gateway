# 调参实验总结

对战发起脚本为 `../find_parameter.py`。
需要事先生成相应 python 文件。
比如，`BASE_POWER, ENEMY_WEIGHT = 3, 1.25` 的文件命名为 `restart_3_125.py`。
原始（被围攻）的参数直接命名为 `restart.py`。
